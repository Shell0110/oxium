Version Alpha de Oxium Clicker

Nous sommes heureux de vous présenter la version Alpha de notre jeux

Qu'est-ce que la version alpha ?

La version alpha est une étape précoce du développement du jeu. Elle est là pour vous faire tester les fonctionnalités de base ainsi que repérer les bugs et recueillir vos premiers retours.

Que pouvez-vous attendre de cette version alpha ?

Dans cette version alpha, vous pouvez vous attendre à :

  • Découvrir les premiers éléments de gameplay comme la fonction de cliquer
  • Explorer un aperçu du monde du jeu
  • Tester les fonctionnalités de base comme l'achat des multiplicateurs
  • Partager vos impressions et suggestions avec l'équipe de développement

Comment participer ?

La participation à la version alpha est limitée du faite que aucune données n'est encore sauvegarder dans notre base données. Restez à l'écoute des annonces officielles sur le discord pour savoir comment participer.

Merci de votre intérêt pour Oxium ! Nous avons hâte de partager cette expérience avec vous.
